Overview

Purpose

My purpose is to create a location for me to display my skills and abilities for showing off to employers. This way I can customize and continually add new material about myself. It will display that I have a diverse knowledge of differing aspects of coding. It would be an amazing place to showcase my current projects as well!

Audience

My audience will be possible future employers. That means for the most part I'll want to keep my project formal. The website will need a clean theme and format to impress those looking at it. While formality is a priority, I'll have to make it unique. If it doesn't stick out then it'll blend in with all the others. So while keeping it formal I'll have to add some aspects that show my personality.

Dynamic elements

I'm planning on having a spotify page because I did a javascript project on one earlier this semester. I'll have a descriptor page that'll have a drop down menu to can the information it displays. Have a contact us page that will allow those who want to contact me can from the page.
